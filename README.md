Staqlt Personnel Roster & Availability Tracker
Setup Instructions

Enable Macros: Ensure the .xlsm file has macros enabled in Trust Center.

Setup Personnel: Use the Personnel_Data sheet to input your core team or use the provided UserForm.

Calendar View: Go to the Calendar tab. Select the year/month in the header to trigger the RefreshCalendar macro.

Qualifications: Edit the Settings tab to add/remove specific certifications.

Features

Dynamic Resource Mapping: Visualizes who is available vs. qualified.

Conflict Detection: Automated warning when a non-qualified staff is assigned.

Office 365 Ready: Scripts included for syncing data to Microsoft Teams or SharePoint.
